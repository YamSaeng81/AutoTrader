## 2. 프로젝트 개요

### 해결하는 문제

가상화폐 시장은 24시간 쉬지 않고 움직인다. 사람이 직접 모든 코인의 차트를 보고 매매 시점을 판단하는 것은 현실적으로 불가능하다. 또한 과거 데이터로 전략을 검증하지 않으면 실제 투자에서 어떤 전략이 얼마나 수익성이 있는지 알 수 없다.

### 솔루션

CryptoAutoTrader는 다음 세 가지를 하나의 플랫폼에서 제공한다:

1. **전략 백테스트**: 2025년 상반기 실제 BTC/ETH 데이터를 기반으로 전략 성과를 검증
2. **모의투자**: 실제 돈을 쓰지 않고 전략의 실시간 성과를 확인
3. **실전매매**: 검증된 전략을 업비트 API와 연동해 자동 실행, 텔레그램으로 매매 알림 수신

### 기대 효과

- 감정 없는 규칙 기반 매매로 충동 매수/매도 방지
- 손절가·익절가 자동 관리로 리스크 통제
- 14종 전략의 시장 국면 자동 선택으로 다양한 시장 상황에 적응
- 웹 대시보드에서 실시간 포지션·수익·로그 확인

## 3. 주요 성과

### ✅ 구현 완료 기능

**백테스팅 엔진**
- 룩어헤드 바이어스 방지, 시장 충격 시뮬레이션, Walk-Forward(과최적화 방지) 포함
- 칼마 비율·최대 낙폭·복구 팩터 등 8종 성과 지표 자동 산출

**전략 14종 (단일 10 + 복합 4)**
- 추세·역추세·변동성·오더북 기반 다양한 전략 포함
- 시장 국면(추세/횡보/고변동성/전환기) 자동 감지 후 최적 전략 선택

**실전매매 시스템**
- 업비트 REST + WebSocket 실시간 연동
- 세션 기반 다중 전략 동시 운영, 최대 낙폭 초과 시 자동 비상정지(Circuit Breaker)
- 텔레그램 실시간 매매 알림 + 일별 요약

**모의투자 시스템**
- 최대 5개 세션 동시 운영, 가상 자금으로 전략 실시간 검증

**웹 대시보드 (16개 페이지)**
- 백테스트 이력·비교, 전략 관리, 모의/실전 매매, 손익 대시보드 등 전 기능 UI 제공

**인프라**
- Docker Compose 운영 배포, Flyway 26단계 DB 버전 관리
- GitHub Actions CI/CD 파이프라인
- Prometheus + Grafana 모니터링
- Spring Security API 토큰 인증

### ❌ 미완료 항목

| 항목 | 영향 | 비고 |
|------|------|------|
| 주간 리포트 API (`GET /api/v1/reports/weekly`) | 낮음 | 텔레그램 알림으로 대체 가능 |
| `COMPOSITE` 범용 전략 타입 백엔드 미등록 | 중간 | `COMPOSITE_BTC`, `COMPOSITE_ETH`로 대체 운영 가능 |
| 독립 `PerformanceController` 클래스 | 낮음 | 기능은 분산 구현으로 정상 동작 |
| `/data` 프론트엔드 페이지 확인 필요 | 낮음 | - |
