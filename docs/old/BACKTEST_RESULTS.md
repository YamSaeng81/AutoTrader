#### 핵심 문제 (미해결)

1. **STOCHASTIC_RSI**: ADX 필터 추가 후에도 -70%. 파라미터 문제가 아니라 전략 구조 자체의 문제일 가능성 높음.
2. **MACD**: ADX > 25 필터에도 -58%. 추세 확인만으로는 부족. 히스토그램 기울기 필터 추가 검토 필요.
3. **EMA_CROSS**: BTC -51% vs ETH +24% — 코인별 특성 차이가 큼. 추세 강도 필터 추가 검토 필요.
4. **VWAP (BTC)**: 승률 0% — 거래 자체가 발생하지 않았거나 모든 거래가 패배. 임계값 2.5%가 너무 높을 수 있음.
